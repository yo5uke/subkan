# SubKan

サブカンは、サブスクの料金・支払日・支払いカードをまとめて管理するアプリです。

Material 3 Expressive に沿ったデザインで、サービス名から自動でアイコンを取得し、月々／年額のサブスクをすっきり一覧表示します。

---

## 📥 ダウンロード

最新版の APK は **[Releases ページ](../../releases/latest)** から入手できます。

> Android 端末向けの `.apk` ファイルです（iOS版はありません）。

### インストール手順（Android）

1. [Releases ページ](../../releases/latest) から `SubKan-vX.Y.Z.apk` をダウンロード
2. ダウンロードした APK ファイルをタップして開く
3. 初回は「**提供元不明のアプリ**」のインストール許可を求められるので、画面の案内に従って許可
   （設定 → アプリ → 不明なアプリのインストール、から許可できます）
4. インストール完了！

## ✨ 主な機能

- **サブスク管理** — サービス名・金額・通貨（JPY / USD / EUR）・支払い間隔（月額 / 年額）・次回支払日・支払いカードを登録、編集、削除
- **アイコン自動取得** — サービス名から公式ロゴを自動表示。取得できない場合も、サービス名から一貫した配色の Material 3 グラデーションタイルを自動生成
- **次回支払日の可視化** — 「あと◯日」「今日」「支払い遅延」を色付きでひと目で確認
- **支払いカード管理** — 複数のカードをブランドカラーで色分けし、各サブスクと紐付け
- **並び替え** — 登録順 / 名前順 / 支払日順、昇順・降順に対応
- **ライト / ダークテーマ** — 端末設定に追従しつつ手動切り替えも可能
- **ローカル保存** — データは端末内にのみ保存。アカウント登録不要で、情報が外部サーバーに送られることはありません

## 🛠️ 開発者向け

Flutter 製です。ソースからビルドする場合:

```bash
flutter pub get
flutter build apk --release
```

リリースは、`v1.0.0` のような**バージョンタグを push すると GitHub Actions が APK を自動ビルドして Releases に添付**します（[ワークフロー](.github/workflows/release.yml)）。

```bash
git tag v1.0.0
git push origin v1.0.0
```
